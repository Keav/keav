Antuuk posted this Insta (Bookmarks) guide on the forum.

Having spent a lot of time actually creating 'Instas' (also known as Bookmarks), to make life easier for all concerned, it is best if everyone adopts the same layout. This way, a squadron can spread out into adjoining systems, and bookmark a large area within a short time. When they come together again and combine the created instas, they are all laid out the same, and all concerned understand them.

1. Ideally, Instas should be created approximately 11 Km from the gate, with the distant gate/station being behind the 11 km object.

2. Instas should be written in the following format - 'X' to 'Y'.
e.g.- if wanting to create an insta from Masanuh to Amarr, it would be laid out thus - 'Masanuh to Amarr'.

3. this might seem obvious, but it allows you also utilise the autopilot indicated route on the top left of your 'ship in space' screen view. Once you jump through a gate, the nearest system, i.e. Masanuh, is indicated. The next gate in line - Amarr - shown in the bottom section, helps you to know that the next insta you are looking for is 'Masanuh to Amarr'.

4. Setting instas for stations - from above or below, get as close to the centre axis of the staion as possible. In most cases (though NOT all), this will allow you to use the station insta from any direction within the system. Stations should also be marked in a certain fashion -

e.g. 'Amarr IV Moon 17 Emperor Family Bureau', should be marked as 'IV Mn17 EFB'.

5. Lastly, created bookmarks should then be placed within a region named folder. This helps when having to copy across created instas for other corp members.
